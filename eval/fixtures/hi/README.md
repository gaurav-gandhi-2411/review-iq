# eval/fixtures/hi — Hindi Eval Fixtures (Synthetic)

6 synthetic Hindi (Devanagari script) fixtures for evaluating extraction from Hindi reviews.

## Source and methodology

These are **synthetic** fixtures — review text and ground truth were both generated by Claude Sonnet (`claude-sonnet-4-5`), then verified by running the production model (Groq Llama 3.3 70B) and confirming score >=85%.

Ground truth was authored then manually corrected for accuracy against the model's actual output behavior (English product names expected in English-output extraction).

**Replace with real Hindi reviews when a buyer demands a Hindi accuracy SLA.**
