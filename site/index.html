        Known gaps:
        Sarcastic Hinglish reviews with backhanded compliments score lower.
        Short informal reviews (<10 words) occasionally miss fields.
        Accuracy on domain-specific jargon (pharma, auto) is untested.
      

    


    
    

      
Integrate in 2 minutes

      
One POST request. JSON in, structured insights out.


      
      

        
        
      


      
      

        

          
curl -X POST https://your-instance.run.app/v2/extract \
  -H "X-API-Key: riq_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{"text": "Excellent product, highly recommend!"}'

          
        

      


      
      

        

          
import httpx

resp = httpx.post(
    "https://your-instance.run.app/v2/extract",
    headers={"X-API-Key": "riq_live_your_key"},
    json={"text": "Excellent product, highly recommend!"},
)
print(resp.json())
